*** Выбор фреймворка ***
В качестве фреймворка выбран Pytest. Это несложный в освоении фреймворк, тесты можно легко масштабировать, в pytest есть
метки (mark), которые позволяют выполнять параметризацию (parametrize) одного и того же тест-кейса,
пропускать его (skip) и т.п. Можно подключить популярные плагины и allure для понятных и красивых отчётов.
В консоли с ним тоже довольно удобно работать.

*** Start ***
1. Устанавливаем пакеты:
pip install -r requirements.txt
2. Запускаем тесты либо в IDE вызовом контекстного окна либо в терминале:
python -m pytest
